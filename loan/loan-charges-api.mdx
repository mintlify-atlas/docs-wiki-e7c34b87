`LoanChargesApiResource` exposes the CRUD-plus-actions surface for `LoanCharge` rows in Apache Fineract. Every charge — fee or penalty, one-off or per-installment — flows through `/v1/loans/{loanId}/charges`. The `?command=` query parameter discriminates between create, pay, waive, adjustment, and deactivate-overdue.

Source: `fineract-provider/src/main/java/org/apache/fineract/portfolio/loanaccount/api/LoanChargesApiResource.java` (541 lines).

For the underlying entity, see [Loan charges](/loan/loan-charges).

```java
@Path("/v1/loans")
@Component
@Tag(name = "Loan Charges", description = …)
@RequiredArgsConstructor
public class LoanChargesApiResource {

    public static final String COMMAND_PAY                  = "pay";
    public static final String COMMAND_WAIVE                = "waive";
    public static final String COMMAND_ADJUSTMENT           = "adjustment";
    public static final String COMMAND_DEACTIVATE_OVERDUE   = "deactivateOverdue";

    private static final Set<String> RESPONSE_DATA_PARAMETERS = new HashSet<>(Arrays.asList(
        "id", "chargeId", "name", "penalty", "chargeTimeType", "dueAsOfDate",
        "chargeCalculationType", "percentage", "amountPercentageAppliedTo", "currency",
        "amountWaived", "amountWrittenOff", "amountOutstanding", "amountOrPercentage",
        "amount", "amountPaid", "chargeOptions", "installmentChargeData", "externalId"));
    private static final String RESOURCE_NAME_FOR_PERMISSIONS = "LOAN";

    private final PlatformSecurityContext        context;
    private final ChargeReadPlatformService      chargeReadPlatformService;
    private final LoanChargeReadPlatformService  loanChargeReadPlatformService;
    private final DefaultToApiJsonSerializer<LoanChargeData> toApiJsonSerializer;
    private final ApiRequestParameterHelper      apiRequestParameterHelper;
    private final PortfolioCommandSourceWritePlatformService commandsSourceWritePlatformService;
    private final LoanReadPlatformService        loanReadPlatformService;
}
```

The `toApiJsonSerializer` is parameterised on `LoanChargeData` (in `fineract-loan/.../data/`).

## Switchboard implementations

### Create / pay-without-id / deactivate-overdue

```java
private String handleExecuteLoanCharge(final Long loanId, final String loanExternalIdStr,
        final String commandParam, final String apiRequestBodyAsJson) {

    ExternalId loanExternalId = ExternalIdFactory.produce(loanExternalIdStr);
    Long resolvedLoanId = loanId == null
        ? loanReadPlatformService.getResolvedLoanId(loanExternalId) : loanId;

    CommandProcessingResult result;
    if (CommandParameterUtil.is(commandParam, COMMAND_PAY)) {
        final CommandWrapper commandRequest = new CommandWrapperBuilder()
            .payLoanCharge(resolvedLoanId, null).withJson(apiRequestBodyAsJson).build();
        result = this.commandsSourceWritePlatformService.logCommandSource(commandRequest);
    } else if (CommandParameterUtil.is(commandParam, COMMAND_DEACTIVATE_OVERDUE)) {
        final CommandWrapper commandRequest = new CommandWrapperBuilder()
            .deactivateOverdueLoanCharges(resolvedLoanId, null).withJson(apiRequestBodyAsJson).build();
        result = this.commandsSourceWritePlatformService.logCommandSource(commandRequest);
    } else {
        final CommandWrapper commandRequest = new CommandWrapperBuilder()
            .createLoanCharge(resolvedLoanId).withJson(apiRequestBodyAsJson).build();
        result = this.commandsSourceWritePlatformService.logCommandSource(commandRequest);
    }
    return this.toApiJsonSerializer.serialize(result);
}
```

### Per-charge pay/waive/adjustment

```java
private String handleExecuteLoanCharge(final Long loanId, final String loanExternalIdStr,
        final Long loanChargeId, final String loanChargeExternalIdStr,
        final String commandParam, final String apiRequestBodyAsJson) {

    ExternalId loanExternalId       = ExternalIdFactory.produce(loanExternalIdStr);
    ExternalId loanChargeExternalId = ExternalIdFactory.produce(loanChargeExternalIdStr);
    Long resolvedLoanId       = loanId == null
        ? loanReadPlatformService.getResolvedLoanId(loanExternalId) : loanId;
    Long resolvedLoanChargeId = getResolvedLoanChargeId(loanChargeId, loanChargeExternalId);

    final CommandWrapperBuilder builder = new CommandWrapperBuilder().withJson(apiRequestBodyAsJson);
    CommandProcessingResult result;
    if (CommandParameterUtil.is(commandParam, COMMAND_WAIVE)) {
        final CommandWrapper commandRequest = builder.waiveLoanCharge(resolvedLoanId, resolvedLoanChargeId).build();
        result = this.commandsSourceWritePlatformService.logCommandSource(commandRequest);
    } else if (CommandParameterUtil.is(commandParam, COMMAND_PAY)) {
        final CommandWrapper commandRequest = builder.payLoanCharge(resolvedLoanId, resolvedLoanChargeId).build();
        result = this.commandsSourceWritePlatformService.logCommandSource(commandRequest);
    } else if (CommandParameterUtil.is(commandParam, COMMAND_ADJUSTMENT)) {
        final CommandWrapper commandRequest = builder.adjustmentForLoanCharge(resolvedLoanId, resolvedLoanChargeId).build();
        result = this.commandsSourceWritePlatformService.logCommandSource(commandRequest);
    } else {
        throw new UnrecognizedQueryParamException("command", commandParam);
    }
    return this.toApiJsonSerializer.serialize(result);
}
```

| `?command=` | Builder verb | Handler bean | Service method |
| --- | --- | --- | --- |
| (none) | `createLoanCharge(loanId)` | `CreateLoanChargeCommandHandler` | `LoanChargeWritePlatformService.addLoanCharge` |
| `pay` (no charge id) | `payLoanCharge(loanId, null)` | `PayLoanChargeCommandHandler` | `LoanChargeWritePlatformService.payLoanCharge` |
| `pay` (with charge id) | `payLoanCharge(loanId, chargeId)` | Same | `payLoanCharge(charge)` |
| `waive` | `waiveLoanCharge(loanId, chargeId)` | `WaiveLoanChargeCommandHandler` | `waiveLoanCharge` |
| `adjustment` | `adjustmentForLoanCharge(loanId, chargeId)` | `AdjustmentForLoanChargeCommandHandler` | `chargeAdjustment` |
| `deactivateOverdue` | `deactivateOverdueLoanCharges(loanId, null)` | `DeactivateOverdueLoanChargesCommandHandler` | `deactivateOverdueLoanCharges` |
| (PUT) | `updateLoanCharge(loanId, chargeId)` | `UpdateLoanChargeCommandHandler` | `updateLoanCharge` |
| (DELETE) | `deleteLoanCharge(loanId, chargeId)` | `DeleteLoanChargeCommandHandler` | `deleteLoanCharge` |

## JSON body shapes

### Create (`POST` no command)

```json
{
  "chargeId": 1,
  "amount": 25.0,
  "dueDate": "20 January 2025",
  "locale": "en",
  "dateFormat": "dd MMMM yyyy",
  "externalId": "fee-001"
}
```

Required: `chargeId`. Optional: `amount` (overrides the charge template default), `dueDate` (required for `SPECIFIED_DUE_DATE`), `externalId`.

If the parent `Charge` is per-installment (`chargeTime = INSTALMENT_FEE`), no `dueDate` is needed — the row fans out across installments via `LoanInstallmentCharge`. See [Loan charges](/loan/loan-charges).

### Pay (`?command=pay`)

```json
{
  "transactionDate": "20 January 2025",
  "locale": "en",
  "dateFormat": "dd MMMM yyyy",
  "paymentTypeId": 1,
  "accountNumber": "…",
  "receiptNumber": "…"
}
```

Materialises a `CHARGE_PAYMENT` (type 17) `LoanTransaction` linked to the charge via `LoanChargePaidBy`.

### Waive (`?command=waive`)

```json
{}
```

No body needed. Waives the entire outstanding amount of the charge (or the specified installment slice for instalment-fees). Creates a `WAIVE_CHARGES` (type 9) `LoanTransaction`.

### Adjustment (`?command=adjustment`)

```json
{
  "transactionDate": "21 January 2025",
  "amount": 15.0,
  "locale": "en",
  "dateFormat": "dd MMMM yyyy",
  "paymentTypeId": 1
}
```

Refunds a portion of a previously-paid charge. Creates a `CHARGE_ADJUSTMENT` (type 26) `LoanTransaction`.

### Deactivate overdue (`?command=deactivateOverdue`)

```json
{
  "transactionDate": "01 February 2025"
}
```

Marks every `OVERDUE_INSTALLMENT` charge on the loan as `active = false`. Used to "cancel" a wave of overdue charges that shouldn't have been applied (e.g. holiday miscount).

### Update (`PUT`)

```json
{
  "amount": 30.0,
  "dueDate": "25 January 2025",
  "locale": "en",
  "dateFormat": "dd MMMM yyyy"
}
```

Only fields supplied are touched. Mutating `amount` or `dueDate` after the loan is disbursed may trigger schedule recalculation if the charge type requires it.

## `retrieveAllLoanCharges` listing

```java
@GET
@Path("{loanId}/charges")
public String retrieveAllLoanCharges(@PathParam("loanId") Long loanId, @Context UriInfo uriInfo) {
    return retrieveAllLoanCharges(loanId, null, uriInfo);
}

private String retrieveAllLoanCharges(Long loanId, String loanExternalIdStr, UriInfo uriInfo) {
    this.context.authenticatedUser().validateHasReadPermission(RESOURCE_NAME_FOR_PERMISSIONS);
    ExternalId loanExternalId = ExternalIdFactory.produce(loanExternalIdStr);
    Long resolvedLoanId = loanId == null ? loanReadPlatformService.getResolvedLoanId(loanExternalId) : loanId;
    Collection<LoanChargeData> loanCharges = this.loanChargeReadPlatformService.retrieveLoanCharges(resolvedLoanId);
    ApiRequestJsonSerializationSettings settings = this.apiRequestParameterHelper.process(uriInfo.getQueryParameters());
    return this.toApiJsonSerializer.serialize(settings, loanCharges, RESPONSE_DATA_PARAMETERS);
}
```

## `retrieveLoanCharge`

```java
@GET
@Path("{loanId}/charges/{loanChargeId}")
public String retrieveLoanCharge(@PathParam("loanId") Long loanId,
        @PathParam("loanChargeId") Long loanChargeId, @Context UriInfo uriInfo) {
    this.context.authenticatedUser().validateHasReadPermission(RESOURCE_NAME_FOR_PERMISSIONS);
    LoanChargeData loanCharge = this.loanChargeReadPlatformService.retrieveLoanChargeDetails(loanChargeId, loanId);
    ApiRequestJsonSerializationSettings settings = this.apiRequestParameterHelper.process(uriInfo.getQueryParameters());
    return this.toApiJsonSerializer.serialize(settings, loanCharge, RESPONSE_DATA_PARAMETERS);
}
```

Throws `LoanChargeNotFoundException` (HTTP 404) if the charge doesn't exist for that loan.

## `retrieveTemplate`

```java
@GET
@Path("{loanId}/charges/template")
public String retrieveTemplate(@PathParam("loanId") Long loanId, @Context UriInfo uriInfo) {
    this.context.authenticatedUser().validateHasReadPermission(RESOURCE_NAME_FOR_PERMISSIONS);
    Collection<ChargeData> loanProductCharges = this.chargeReadPlatformService.retrieveLoanAccountApplicableCharges(loanId,
        new ChargeTimeType[] { ChargeTimeType.OVERDUE_INSTALLMENT });
    final LoanChargeData newLoanChargeData = LoanChargeData.template(loanProductCharges);
    final ApiRequestJsonSerializationSettings settings = this.apiRequestParameterHelper.process(uriInfo.getQueryParameters());
    return this.toApiJsonSerializer.serialize(settings, newLoanChargeData, RESPONSE_DATA_PARAMETERS);
}
```

Note that `OVERDUE_INSTALLMENT` charges are excluded from the template — they are not manually addable; they are applied by the COB step. See [Loan COB business steps](/cob/loan-cob-business-steps).

## External-id resolution helper

```java
private Long getResolvedLoanChargeId(Long loanChargeId, ExternalId loanChargeExternalId) {
    return loanChargeId == null
        ? loanChargeReadPlatformService.resolveLoanChargeId(loanChargeExternalId)
        : loanChargeId;
}
```
