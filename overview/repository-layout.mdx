This page enumerates every directory at the root of the Apache Fineract repository (`/`) together with the `src/main/java/...` package root of each Gradle source module. The Gradle module list is the one declared in `settings.gradle` (`rootProject.name='fineract'`); the dynamic `custom/<company>/<category>/<module>` loader is described in [`/overview/module-graph`](/overview/module-graph).

## Top-level directories

The table is exhaustive. Files (e.g. `README.md`, `build.gradle`, `gradlew`) are listed in the next section.

| Path | Type | Role |
|------|------|------|
| `.github/` | meta | GitHub Actions workflows and ASF issue templates. |
| `.theia/` | meta | Eclipse Theia / Gitpod IDE settings. |
| `buildSrc/` | gradle | Shared Gradle precompiled scripts; `buildSrc/src/main/groovy/org.apache.fineract.dependencies.gradle` defines version catalogs consumed by `fineract-war/build.gradle` and others. |
| `config/` | tooling | Checkstyle, SpotBugs and Eclipse formatter config: `config/checkstyle/`, `config/spotbugs/`, `config/fineractdev-formatter.xml`. |
| `config/docker/` | tooling | Static configs used by the docker image build. |
| `custom/` | extension | Dynamically-loaded custom modules under `custom/<company>/<category>/<module>` plus `custom/docker/` for the custom Docker image; loader logic in `settings.gradle`. |
| `docker/` | tooling | Tomcat `server.xml` used by the WAR-based docker image. |
| `fineract-accounting/` | source module | General ledger, journal entries, accruals, provisioning. |
| `fineract-avro-schemas/` | source module | Avro IDL/`.avsc` schemas for external events. |
| `fineract-branch/` | source module | Branch organisation, tellers, cash drawers. |
| `fineract-charge/` | source module | Charges and fees. |
| `fineract-client/` | source module | Generated REST client (Java) — `swagger-codegen` output. |
| `fineract-client-feign/` | source module | Spring Cloud OpenFeign variant of the client. |
| `fineract-cob/` | source module | Close-of-Business job framework, business-step pipeline. |
| `fineract-command/` | source module | Command-source persistence and core command APIs. |
| `fineract-core/` | source module | Cross-cutting infrastructure: tenants, security model, configuration, common entities. |
| `fineract-db/` | data | Tenant-master and demo MariaDB/PostgreSQL backups: `fineract-db/multi-tenant-demo-backups/`, plus legacy SQL in `fineract-db/old-schema-files/`. |
| `fineract-doc/` | docs | Antora/AsciiDoc developer documentation source (`fineract-doc/modules/ROOT/pages/`). |
| `fineract-document/` | source module | Document attachment store (filesystem + S3) and content type whitelisting. |
| `fineract-e2e-tests-core/` | tests | Cucumber framework and Karate steps reused by `fineract-e2e-tests-runner`. |
| `fineract-e2e-tests-runner/` | tests | Runnable end-to-end test harness. |
| `fineract-investor/` | source module | External-asset-owner / investor capital-flow tracking. |
| `fineract-loan/` | source module | Classic (non-progressive) loan domain — products, accounts, schedules. |
| `fineract-loan-origination/` | source module | Loan application / origination workflow. |
| `fineract-mix/` | source module | MIX Market XBRL taxonomy mapping and reporting. |
| `fineract-progressive-loan/` | source module | Newer progressive-strategy loan engine that shares some entities with `fineract-loan`. |
| `fineract-progressive-loan-embeddable-schedule-generator/` | source module | Standalone JAR that exposes the progressive-loan schedule generator as an embeddable library. |
| `fineract-provider/` | source module | Composition root — `ServerApplication.java` lives here; depends on every feature module. |
| `fineract-rates/` | source module | Floating interest rates. |
| `fineract-report/` | source module | Reporting infrastructure (Pentaho, JasperReports, ad-hoc SQL). |
| `fineract-savings/` | source module | Savings products, deposits, fixed-deposit accounts. |
| `fineract-security/` | source module | Spring Security, OAuth2, 2FA, SQL-injection guards. |
| `fineract-tax/` | source module | Tax groups and tax components. |
| `fineract-validation/` | source module | Custom Jakarta validators and ICU4j-backed bean validation. |
| `fineract-war/` | packaging | Assembles `fineract-provider.war` for Tomcat deployment. |
| `fineract-working-capital-loan/` | source module | Working-capital-loan product variant on top of `fineract-loan`. |
| `gradle/` | build | Gradle wrapper jar + properties in `gradle/wrapper/`. |
| `integration-tests/` | tests | Black-box integration tests against a running Fineract instance. |
| `kubernetes/` | tooling | Kubernetes manifests (`fineract-server-deployment.yml`, `fineractmysql-*.yml`) and helper scripts (`kubectl-startup.sh`, `kubectl-shutdown.sh`). |
| `oauth2-tests/` | tests | OAuth2-specific integration tests. |
| `scripts/` | tooling | Maintenance scripts (`jpa-scanner-grouped.sh`, `split-features.sh`, `split-tests.sh`, `verify-signed-commits.sh`). |
| `twofactor-tests/` | tests | Integration tests for the 2FA pipeline. |

| `README.md` | High-level project README. |
| `renovate.json` | Renovate-bot configuration. |

| `.profileconfig.json` | IDE profile metadata. |

| `docker-compose.yml` | Default compose stack (provider + mariadb). |
| `docker-compose-mariadb.yml` | MariaDB-backed stack. |
| `docker-compose-mysql.yml` | MySQL-backed stack. |
| `docker-compose-postgresql.yml` | PostgreSQL-backed stack. |

| `docker-compose-postgresql-kafka-msk.yml` | Same but tuned for AWS MSK. |

| `docker-compose-community-app.yml` | Stack including the Mifos community-app frontend. |
| `docker-compose-web-app.yml` | Stack with the legacy web-app frontend. |
| `docker-compose-custom.yml` | Stack that includes `custom/docker` built image. |

## Source module — `fineract-provider`

`fineract-provider` is the composition root. Its `src/main/java/org/apache/fineract/` tree mixes feature folders that pre-date the modular split with cross-cutting `infrastructure/` packages.

| Package root | Role |
|--------------|------|
| `org/apache/fineract/ServerApplication.java` | Spring Boot main class. |
| `org/apache/fineract/accounting/` | Provider-side accounting glue (`accrual`, `journalentry`, `provisioning`, `jobs`, `productaccountmapping`). |
| `org/apache/fineract/adhocquery/` | Ad-hoc SQL query domain — `api`, `data`, `domain`, `handler`, `service`, `starter`. |
| `org/apache/fineract/batch/` | HTTP batch executor (multiple REST calls in one request). |
| `org/apache/fineract/cob/` | Provider-side COB wiring (`api`, `loan`, `savings`, `service`, `listener`, `conditions`). |
| `org/apache/fineract/commands/` | Provider-side command APIs (`api`, `data`, `service`, `starter`). |
| `org/apache/fineract/infrastructure/accountnumberformat/` | Configurable account-number generation. |
| `org/apache/fineract/infrastructure/bulkimport/` | Bulk Excel import handlers. |
| `org/apache/fineract/infrastructure/campaigns/` | SMS/email outbound campaigns. |
| `org/apache/fineract/infrastructure/codes/` | Code-value lookups. |
| `org/apache/fineract/infrastructure/configuration/` | `c_configuration` global flags. |
| `org/apache/fineract/infrastructure/core/` | Provider-specific core overrides on top of `fineract-core`. |
| `org/apache/fineract/infrastructure/creditbureau/` | Credit-bureau integration. |
| `org/apache/fineract/infrastructure/dataqueries/` | Datatables and stretchy reports. |
| `org/apache/fineract/infrastructure/entityaccess/` | Entity-to-office access mapping. |
| `org/apache/fineract/infrastructure/event/` | Internal `BusinessEvent` listeners. |
| `org/apache/fineract/infrastructure/gcm/` | Mobile push (Google Cloud Messaging). |
| `org/apache/fineract/infrastructure/hooks/` | Outbound webhooks. |
| `org/apache/fineract/infrastructure/instancemode/` | Runtime instance-mode REST API (test-profile only). |
| `org/apache/fineract/infrastructure/jobs/` | Scheduled-job registry. |
| `org/apache/fineract/infrastructure/reportmailingjob/` | Scheduled report email delivery. |
| `org/apache/fineract/infrastructure/security/` | Provider-specific security extensions. |
| `org/apache/fineract/infrastructure/sms/` | SMS gateway integration. |
| `org/apache/fineract/infrastructure/springbatch/` | `ManagerConfig`, `WorkerConfig`, JMS/Kafka conditions. |
| `org/apache/fineract/infrastructure/survey/` | Survey / poverty index. |
| `org/apache/fineract/interoperation/` | Mojaloop interoperability (`api`, `data`, `domain`, `handler`). |
| `org/apache/fineract/notification/` | User-notification system (`fineract.notification.user-notification-system.enabled`). |
| `org/apache/fineract/organisation/` | Office, staff, holiday, working-day, currency, monetary entities. |
| `org/apache/fineract/portfolio/` | Client, group, fund, account, charge, savings, loan-account portfolio domain. |
| `org/apache/fineract/spm/` | Social Performance Management surveys. |
| `org/apache/fineract/template/` | Mustache-style document templates (`fineract.template.regex-whitelist`). |
| `org/apache/fineract/useradministration/` | App-user, role, permission CRUD. |

Resources live under `fineract-provider/src/main/resources/`:

| Path | Role |
|------|------|
| `application.properties` | All `fineract.*` defaults. |
| `db/changelog/` | Liquibase master + tenant changelogs. |
| `META-INF/persistence.xml` | EclipseLink persistence unit. |
| `static/` | Static OpenAPI UI assets. |
