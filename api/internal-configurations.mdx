The Internal Configurations API is a **test-profile-only** trap-door that Apache Fineract integration tests use to bypass maker-checker, validation, and the normal `UPDATE_CONFIGURATION` permission and directly mutate a `GlobalConfigurationProperty` row. The resource is registered only when the Spring profile `FineractProfiles.TEST` is active and writes a loud `DO NOT USE THIS IN PRODUCTION!` warning to the log at startup.

## Source

| Aspect | Value |
| --- | --- |
| Resource class | `org.apache.fineract.infrastructure.configuration.api.InternalConfigurationsApiResource` |
| File | `fineract-provider/src/main/java/org/apache/fineract/infrastructure/configuration/api/InternalConfigurationsApiResource.java` |
| JAX-RS `@Path` | `/v1/internal/configurations` |
| Active profile | `FineractProfiles.TEST` |
| Repository | `GlobalConfigurationRepositoryWrapper` |
| Side effect | `MoneyHelperInitializationService.initializeTenantRoundingMode(tenant)` when `configName == ROUNDING_MODE` |

The class also implements `InitializingBean` purely so that the warning banner runs at context startup.

## Endpoints

| Method | Path | Description | Command handler | Permission |
| --- | --- | --- | --- | --- |
| `PUT` | `/v1/internal/configurations/name/{configName}/value/{configValue}` | Directly set `GlobalConfigurationProperty.value` for the named config. | Inline repository mutation; no `CommandWrapper` is built. | None — gated by the `TEST` Spring profile only |

The implementation:

1. `repository.findOneByNameWithNotFoundDetection(configName)` — 404 if the name is unknown.
2. `config.setValue(configValue)` and `repository.save(config)`.
3. `repository.removeFromCache(config.getName())` to invalidate the cached configuration property.
4. When `configName == GlobalConfigurationConstants.ROUNDING_MODE`, `MoneyHelperInitializationService` is asked to re-bind the tenant's rounding mode immediately.

## Path parameters

| Name | Type | Notes |
| --- | --- | --- |
| `configName` | string | Name of an existing global configuration row. |
| `configValue` | long | Numeric value to assign; non-numeric configurations are not supported here. |

## Response

`Response.status(OK).build()` — empty body.

## Usage in tests

Typical use from a JUnit integration test:

```http
PUT /fineract-provider/api/v1/internal/configurations/name/rounding-mode/value/4
```

flipping `RoundingMode.ordinal()` (`4` = `HALF_UP`). The test then continues with the rest of its scenario without having to authenticate as an admin user or wait for maker-checker.

## Production gating

In any non-test profile (default, production, etc.) the Spring `@Profile(FineractProfiles.TEST)` annotation prevents bean creation, so the JAX-RS path simply returns 404. Operators must not enable the test profile in production.
