`WorkingCapitalLoanApiResource` is the entry-point JAX-RS resource for Apache Fineract's **Working Capital Loan (WCL)** product family. WCL is a separate product type from the classic `Loan` — it has its own product, account, transactions and amortization-schedule models. The resource lives in the dedicated `fineract-working-capital-loan` module so the build profile can exclude it.

This page documents only the **application lifecycle**: list / retrieve, submit, modify, delete, and state transitions (approve / reject / undo). Transaction endpoints are documented on [working-capital-loan-transactions](/api/working-capital-loan-transactions); the projected amortization schedule on [working-capital-loan-amortization-schedule](/api/working-capital-loan-amortization-schedule); and product CRUD on [working-capital-loan-products](/api/working-capital-loan-products).

## Endpoints

### Read

| Method | Path | Description | Command / Handler | Permission |
| ------ | ---- | ----------- | ----------------- | ---------- |
| GET | `/v1/working-capital-loans/template?productId=&clientId=` | Return application template (product options, fund options, delinquency bucket options, period frequency types). | `readPlatformService.retrieveTemplate(productId, clientId)` | Read permission `WORKINGCAPITALLOAN` |
| GET | `/v1/working-capital-loans?externalId=&accountNo=&clientId=&status=&page=&size=&sort=` | Spring-Data paged list. Filter by `clientId`, `externalId`, `status`, `accountNo`. | `readPlatformService.retrieveAllPaged(...)` | Read permission `WORKINGCAPITALLOAN` |
| GET | `/v1/working-capital-loans/{loanId}` | Retrieve by internal id. | `readPlatformService.retrieveOne(loanId)` | Read permission `WORKINGCAPITALLOAN` |
| GET | `/v1/working-capital-loans/external-id/{loanExternalId}` | Retrieve by external id. | `readPlatformService.retrieveOne(externalId)` | Read permission `WORKINGCAPITALLOAN` |

The `Pageable` is bound via the platform-specific `@Pagination(maximumSize = 200)` parameter resolver.

### Write — application lifecycle

| Method | Path | Description | Command / Handler | Permission |
| ------ | ---- | ----------- | ----------------- | ---------- |
| POST | `/v1/working-capital-loans` | Submit a new WCL application. | `CommandWrapperBuilder.createWorkingCapitalLoanApplication()` | `CREATE_WORKINGCAPITALLOAN` |
| PUT | `/v1/working-capital-loans/{loanId}` | Modify a WCL application (only valid in `Submitted and pending approval` status). | `CommandWrapperBuilder.withLoanId(loanId).updateWorkingCapitalLoanApplication()` | `UPDATE_WORKINGCAPITALLOAN` |
| PUT | `/v1/working-capital-loans/external-id/{loanExternalId}` | Same modify, keyed by external id. | Same as above; loan id resolved via `readPlatformService.getResolvedLoanId(externalId)` | `UPDATE_WORKINGCAPITALLOAN` |
| DELETE | `/v1/working-capital-loans/{loanId}` | Delete an application — only valid in `Submitted and awaiting approval` status. | `CommandWrapperBuilder.deleteWorkingCapitalLoanApplication()` | `DELETE_WORKINGCAPITALLOAN` |
| DELETE | `/v1/working-capital-loans/external-id/{loanExternalId}` | Same delete, keyed by external id. | As above | `DELETE_WORKINGCAPITALLOAN` |

### Write — state transitions

| Method | Path | `command` query value | Description | Command / Handler |
| ------ | ---- | --------------------- | ----------- | ----------------- |
| POST | `/v1/working-capital-loans/{loanId}?command=approve` | `approve` | Approve the application. | `approveWorkingCapitalLoanApplication(resolvedLoanId)` |
| POST | `/v1/working-capital-loans/{loanId}?command=reject` | `reject` | Reject the application. | `rejectWorkingCapitalLoanApplication(resolvedLoanId)` (see `handleStateTransition` in source) |
| POST | `/v1/working-capital-loans/{loanId}?command=undoapproval` | `undoapproval` | Undo a prior approval. | `undoApprovalWorkingCapitalLoanApplication(resolvedLoanId)` |
| POST | `/v1/working-capital-loans/external-id/{loanExternalId}?command=…` | – | Same as above, keyed by external id. | – |

The `command` query parameter is **mandatory** on transition endpoints; an unrecognised value throws `UnrecognizedQueryParamException`. The handler is `handleStateTransition` in the source:

```java
if (CommandParameterUtil.is(commandParam, "approve")) {
    commandRequest = builder.approveWorkingCapitalLoanApplication(resolvedLoanId).build();
} else if (CommandParameterUtil.is(commandParam, "reject")) {
    // ...
}
```

## Request / response examples

### Template

`GET /v1/working-capital-loans/template?productId=12&clientId=55`

```json
{
  "clientId": 55,
  "productId": 12,
  "productName": "Standard WCL",
  "currency": { "code": "USD", "decimalPlaces": 2 },
  "productOptions": [ { "id": 12, "name": "Standard WCL" } ],
  "fundOptions": [],
  "delinquencyBucketOptions": [],
  "periodFrequencyTypeOptions": [
    { "id": 0, "code": "frequency.days",   "value": "Days" },
    { "id": 2, "code": "frequency.months", "value": "Months" }
  ]
}
```

### Submit application

`POST /v1/working-capital-loans`

```json
{
  "clientId": 55,
  "productId": 12,
  "principal": 10000.00,
  "periodPaymentRate": 0.10,
  "repaymentEvery": 30,
  "repaymentFrequencyType": 0,
  "submittedOnDate": "01 January 2025",
  "expectedDisbursementDate": "05 January 2025",
  "locale": "en",
  "dateFormat": "dd MMMM yyyy"
}
```

```json
{
  "clientId": 55,
  "loanId": 1001,
  "resourceId": 1001,
  "changes": {}
}
```


## Data carriers

- **Requests:** raw JSON; the resource passes the body straight into the `CommandWrapper`. Swagger types in `WorkingCapitalLoanApiResourceSwagger`:
  - `PostWorkingCapitalLoansRequest`, `PostWorkingCapitalLoansLoanIdRequest`
  - `PutWorkingCapitalLoansLoanIdRequest`
- **Read responses:** `WorkingCapitalLoanData`, `WorkingCapitalLoanTemplateData`. Paged via Spring `Page<>`.
- **Write responses:** `CommandProcessingResult` envelope (Swagger: `PostWorkingCapitalLoansResponse`, `PutWorkingCapitalLoansLoanIdResponse`, `DeleteWorkingCapitalLoansLoanIdResponse`, `PostWorkingCapitalLoansLoanIdResponse`).

## Permissions

All read endpoints call:

```java
this.context.authenticatedUser().validateHasReadPermission(RESOURCE_NAME_FOR_PERMISSIONS);
```

Writes delegate authorization to `PortfolioCommandSourceWritePlatformService`.
