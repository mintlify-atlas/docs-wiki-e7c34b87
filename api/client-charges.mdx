`ClientChargesApiResource` is the entry point for fees that are levied directly on a client (not on a loan or savings account). Clients can be charged ad-hoc — for example membership fees, registration fees, account-maintenance fees — and the system tracks whether each charge is paid, waived, or still outstanding.

## Permissions

Resource string: `CLIENTCHARGE` (`ClientApiConstants.CLIENT_CHARGES_RESOURCE_NAME`). Every read calls `validateHasReadPermission("CLIENTCHARGE")`. Writes are gated by `CREATE_CLIENTCHARGE`, `PAY_CLIENTCHARGE`, `WAIVE_CLIENTCHARGE`, `INACTIVATE_CLIENTCHARGE`, `DELETE_CLIENTCHARGE`.

## Endpoint inventory

| HTTP | Path | Description | Command / Read service |
| --- | --- | --- | --- |
| `GET` | `/v1/clients/{clientId}/charges` | Paged list, filtered by status | `clientChargeReadPlatformService.retrieveClientCharges(...)` |
| `GET` | `/v1/clients/{clientId}/charges/template` | Eligible charge definitions for this client | `clientChargeReadPlatformService.retrieveClientChargeTemplate(clientId)` |
| `GET` | `/v1/clients/{clientId}/charges/{chargeId}` | Fetch one charge | `clientChargeReadPlatformService.retrieveClientCharge(clientId, chargeId)` |
| `POST` | `/v1/clients/{clientId}/charges` | Apply a charge | `createClientCharge(clientId)` |
| `POST` | `/v1/clients/{clientId}/charges/{chargeId}?command=paycharge` | Pay the charge (constant `CLIENT_CHARGE_COMMAND_PAY_CHARGE`) | `payClientCharge(clientId, chargeId)` |
| `POST` | `/v1/clients/{clientId}/charges/{chargeId}?command=waive` | Waive the charge | `waiveClientCharge(clientId, chargeId)` |
| `POST` | `/v1/clients/{clientId}/charges/{chargeId}?command=inactivate` | Mark inactive (no further demand) | `inactivateClientCharge(clientId, chargeId)` |
| `DELETE` | `/v1/clients/{clientId}/charges/{chargeId}` | Delete a charge | `deleteClientCharge(clientId, chargeId)` |

## Source excerpt — multi-verb dispatch

```java
@POST
@Path("{chargeId}")
public String payOrWaiveClientCharge(
        @PathParam("clientId") final Long clientId,
        @PathParam("chargeId") final Long chargeId,
        @QueryParam("command") final String commandParam,
        final String apiRequestBodyAsJson) {

    if (is(commandParam, "waive")) {
        final CommandWrapper cw = new CommandWrapperBuilder()
            .waiveClientCharge(clientId, chargeId)
            .withJson(apiRequestBodyAsJson).build();
        return toApiJsonSerializer.serialize(
            commandsSourceWritePlatformService.logCommandSource(cw));
    } else if (is(commandParam, "paycharge")) {
        final CommandWrapper cw = new CommandWrapperBuilder()
            .payClientCharge(clientId, chargeId)
            .withJson(apiRequestBodyAsJson).build();
        return toApiJsonSerializer.serialize(
            commandsSourceWritePlatformService.logCommandSource(cw));
    } else if (is(commandParam, "inactivate")) {
        final CommandWrapper cw = new CommandWrapperBuilder()
            .inactivateClientCharge(clientId, chargeId).build();
        return toApiJsonSerializer.serialize(
            commandsSourceWritePlatformService.logCommandSource(cw));
    }
    throw new UnrecognizedQueryParamException("command", commandParam);
}
```

## Canonical curl

Apply a charge:

```bash
curl -k -u mifos:password \
  -H "Fineract-Platform-TenantId: default" \
  -H "Content-Type: application/json" \
  -X POST https://localhost:8443/fineract-provider/api/v1/clients/187/charges \
  -d '{
    "chargeId": 8,
    "amount": 100.0,
    "dueDate": "04 March 2024",
    "locale": "en",
    "dateFormat": "dd MMMM yyyy"
  }'
```

Pay the charge:

```bash
curl -k -u mifos:password \
  -H "Fineract-Platform-TenantId: default" \
  -H "Content-Type: application/json" \
  -X POST 'https://localhost:8443/fineract-provider/api/v1/clients/187/charges/55?command=paycharge' \
  -d '{
    "transactionDate": "05 March 2024",
    "amount": 100.0,
    "locale": "en",
    "dateFormat": "dd MMMM yyyy"
  }'
```

## Request body shapes

### POST (apply charge)

| Field | Required | Notes |
| --- | --- | --- |
| `chargeId` | yes | id of a `Charge` definition with `chargeAppliesTo = CLIENT` |
| `amount` | no | overrides the default on the charge |
| `dueDate` | yes | `dd MMMM yyyy` (per `dateFormat`) |
| `locale` | yes | e.g. `en` |
| `dateFormat` | yes | e.g. `dd MMMM yyyy` |

### POST `?command=pay`

| Field | Required | Notes |
| --- | --- | --- |
| `amount` | yes | partial payments allowed |
| `transactionDate` | yes | |
| `paymentTypeId` | no | links to `m_payment_type` |
| `accountNumber`, `chequeNumber`, `routingCode`, `receiptNumber`, `bankNumber` | no | payment-detail fields |
| `locale`, `dateFormat` | yes | |

### POST `?command=waive`

| Field | Required | Notes |
| --- | --- | --- |
| (none beyond `locale`, `dateFormat`) | | Waives the outstanding amount |
