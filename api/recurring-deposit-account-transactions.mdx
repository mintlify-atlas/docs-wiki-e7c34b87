The Recurring Deposit Account Transactions API records periodic deposits and (if the product permits) withdrawals on Apache Fineract recurring-deposit accounts, plus the standard undo / modify corrections.

## Source

| Aspect | Value |
| --- | --- |
| Resource class | `org.apache.fineract.portfolio.savings.api.RecurringDepositAccountTransactionsApiResource` |
| File | `fineract-provider/src/main/java/org/apache/fineract/portfolio/savings/api/RecurringDepositAccountTransactionsApiResource.java` |
| JAX-RS `@Path` | `/v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions` |
| Swagger tag | `Recurring Deposit Account Transactions` |
| Permission resource | `RECURRINGDEPOSITACCOUNT` |
| Read service | `SavingsAccountReadPlatformService` |
| Command source | `PortfolioCommandSourceWritePlatformService` |

## Endpoints

| Method | Path | Operation id | Description |
| --- | --- | --- | --- |
| `GET` | `/v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions/template` | `retrieveTemplateRecurringDepositAccountTransaction` | Transaction template (requires `?command=deposit` or `?command=withdrawal`). |
| `GET` | `/v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions/{transactionId}` | `retrieveOneRecurringDepositAccountTransaction` | Retrieve a single transaction. |
| `POST` | `/v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions` | `transactionRecurringDepositAccountTransaction` | Execute a deposit or withdrawal. |
| `POST` | `/v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions/{transactionId}` | (`handleTransactionCommands`) | Undo or modify an existing transaction. |

### Template

The template endpoint requires the `?command=` query parameter:

```java
if (!(is(commandParam, "deposit") || is(commandParam, "withdrawal"))) {
    throw new UnrecognizedQueryParamException("command", commandParam,
        new Object[] { "deposit", "withdrawal" });
}
```

## Transaction commands

`POST /v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions?command={cmd}`:

| `command` | Builder |
| --- | --- |
| `deposit` | `recurringAccountDeposit(recurringDepositAccountId)` |
| `withdrawal` | `recurringAccountWithdrawal(recurringDepositAccountId)` |

Any other value raises `UnrecognizedQueryParamException("command", commandParam, "deposit", "withdrawal")`.

## Adjust commands

`POST /v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions/{transactionId}?command={cmd}`:

| `command` | Builder |
| --- | --- |
| `undo` (`DepositsApiConstants.COMMAND_UNDO_TRANSACTION`) | `undoRecurringAccountTransaction(recurringDepositAccountId, transactionId)` |
| `modify` (`DepositsApiConstants.COMMAND_ADJUST_TRANSACTION`) | `adjustRecurringAccountTransaction(recurringDepositAccountId, transactionId)` |

Any other value raises `UnrecognizedQueryParamException("command", commandParam, "undo", "modify")`.

## Request shapes

### Deposit

`POST /v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions?command=deposit`:

```json
{
  "transactionDate": "01 April 2026",
  "transactionAmount": 200,
  "paymentTypeId": 1,
  "locale": "en",
  "dateFormat": "dd MMMM yyyy"
}
```

### Withdrawal (when allowed by the product)

`POST /v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions?command=withdrawal`:

```json
{
  "transactionDate": "01 May 2026",
  "transactionAmount": 50,
  "paymentTypeId": 1,
  "locale": "en",
  "dateFormat": "dd MMMM yyyy"
}
```

### Undo

`POST /v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions/{transactionId}?command=undo` accepts an empty body.

### Modify

`POST /v1/recurringdepositaccounts/{recurringDepositAccountId}/transactions/{transactionId}?command=modify`:

```json
{
  "transactionDate": "01 April 2026",
  "transactionAmount": 250,
  "paymentTypeId": 1,
  "locale": "en",
  "dateFormat": "dd MMMM yyyy"
}
```

## Permissions

Read endpoints invoke `validateHasReadPermission("RECURRINGDEPOSITACCOUNT")`. Writes go through `PortfolioCommandSourceWritePlatformService.logCommandSource(...)` and are mapped to `DEPOSIT_RECURRINGDEPOSITACCOUNT`, `WITHDRAWAL_RECURRINGDEPOSITACCOUNT`, `UNDOTRANSACTION_RECURRINGDEPOSITACCOUNT`, `ADJUSTTRANSACTION_RECURRINGDEPOSITACCOUNT`.
