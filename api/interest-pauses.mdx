The Loan Interest Pauses API in Apache Fineract lets operators carve out a period in which no interest will accrue on a loan — for example during a documented hardship, disaster moratorium or contractual grace. Each pause is stored as a `LoanTermVariation` of type `Interest Pause` and can be created, listed, updated, and deleted via the `LoanInterestPauseApiResource`.

## Endpoints

### By loan id

| Method | Path | Description | Command / read handler |
| --- | --- | --- | --- |
| `POST` | `/v1/loans/{loanId}/interest-pauses` | Create a new interest-pause period. | `CommandWrapperBuilder.createInterestPause(loanId)` |
| `GET` | `/v1/loans/{loanId}/interest-pauses` | List every active interest-pause on the loan. | `InterestPauseReadPlatformService.retrieveInterestPauses(loanId)` |
| `PUT` | `/v1/loans/{loanId}/interest-pauses/{variationId}` | Update a specific interest-pause (by `LoanTermVariations.id`). | `CommandWrapperBuilder.updateInterestPause(loanId, variationId)` |
| `DELETE` | `/v1/loans/{loanId}/interest-pauses/{variationId}` | Delete a specific interest-pause. Returns HTTP 204. | `CommandWrapperBuilder.deleteInterestPause(loanId, variationId)` |

### By loan external id

| Method | Path | Mirror of |
| --- | --- | --- |
| `POST` | `/v1/loans/external-id/{loanExternalId}/interest-pauses` | `POST /v1/loans/{loanId}/interest-pauses` |
| `GET` | `/v1/loans/external-id/{loanExternalId}/interest-pauses` | `GET /v1/loans/{loanId}/interest-pauses` |
| `PUT` | `/v1/loans/external-id/{loanExternalId}/interest-pauses/{variationId}` | `PUT /v1/loans/{loanId}/interest-pauses/{variationId}` |
| `DELETE` | `/v1/loans/external-id/{loanExternalId}/interest-pauses/{variationId}` | `DELETE /v1/loans/{loanId}/interest-pauses/{variationId}` |

The external-id variants resolve the loan internally and reuse the same command builders (`createInterestPauseByExternalId`, `updateInterestPause(externalId, variationId)`, `deleteInterestPause(externalId, variationId)`).

## Request / response shapes

### Create / update

Body type `InterestPauseRequestDto`:

```json
{
  "startDate": "01 June 2026",
  "endDate": "31 August 2026",
  "locale": "en",
  "dateFormat": "dd MMMM yyyy"
}
```

`startDate` and `endDate` define the inclusive period. The server validates that the range falls within the loan's schedule and does not overlap an existing interest pause.

### List response

`GET /v1/loans/{loanId}/interest-pauses` returns a `List<InterestPauseResponseDto>`:

```json
[
  {
    "variationId": 12,
    "loanId": 123,
    "startDate": "2026-06-01",
    "endDate": "2026-08-31",
    "isActive": true
  }
]
```

### Create / update response

`CommandProcessingResult`:

```json
{
  "officeId": 1,
  "clientId": 42,
  "loanId": 123,
  "resourceId": 12,
  "changes": { "startDate": "2026-06-01", "endDate": "2026-08-31" }
}
```

### Delete response

The delete endpoints return `Response.noContent().build()` (HTTP 204) regardless of internal id or external id variant.

## Permissions

The resource exposes two constants:

```java
private static final String RESOURCE_NAME_FOR_PERMISSIONS        = "LOAN";
private static final String MODIFY_RESOURCE_NAME_FOR_PERMISSIONS = "UPDATE LOAN";
```

- `GET` calls invoke `validateHasReadPermission("LOAN")`.
- `POST`, `PUT`, `DELETE` invoke `validateHasReadPermission("UPDATE LOAN")` (note the space in the resource name — that is the actual production code).

After the read check, writes are logged through `PortfolioCommandSourceWritePlatformService.logCommandSource(...)` which applies the per-command permission mapped from `createInterestPause` / `updateInterestPause` / `deleteInterestPause`.

## Behaviour notes

- An interest pause is implemented as a `LoanTermVariations` row of type `INTEREST_PAUSE`. Subsequent reschedule operations preserve the variation but recompute future installments.
- For progressive loans, the schedule generator skips the daily-interest accrual for any installment falling fully inside the pause window.
- Deleting a pause re-applies the standard accrual rules going forward; historical accrual entries remain untouched.
