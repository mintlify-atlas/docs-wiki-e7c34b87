The Delinquency API is the configuration entry-point that Apache Fineract uses to classify overdue loans. A `DelinquencyRange` is a bucket of past-due days with classification metadata; a `DelinquencyBucket` groups multiple ranges and is attached to a `LoanProduct`. `DelinquencyApiResource` exposes CRUD for both, plus a template that returns every dropdown a UI needs to author a bucket.

## Template composition

`GET /v1/delinquency/buckets/template` aggregates everything a UI needs to build a bucket:

- `DelinquencyResponseMapper.mapRange(retrieveAllDelinquencyRanges())` — selectable ranges
- `DelinquencyBucketType.toStringEnumOptions()` — bucket type enum
- `DelinquencyFrequencyType.toStringEnumOptions()` — frequency enum (`days`, `weeks`, `months`)
- `DelinquencyMinimumPaymentType.toStringEnumOptions()` — minimum-payment enum

```json
{
  "delinquencyRangeOptions": [
    { "id": 1, "classification": "Range 1-30", "minimumAgeDays": 1, "maximumAgeDays": 30 }
  ],
  "delinquencyBucketTypeOptions": [ { "code": "delinquencyBucketType.basic" } ],
  "delinquencyFrequencyTypeOptions": [
    { "code": "delinquencyFrequencyType.days" },
    { "code": "delinquencyFrequencyType.weeks" }
  ],
  "delinquencyMinimumPaymentTypeOptions": [
    { "code": "delinquencyMinimumPaymentType.installment" }
  ]
}
```

## Request shapes

### Create a range

`POST /v1/delinquency/ranges` (body type `DelinquencyRangeRequest`):

```json
{
  "classification": "Range 1-30",
  "minimumAgeDays": 1,
  "maximumAgeDays": 30,
  "locale": "en"
}
```

### Update a range

`PUT /v1/delinquency/ranges/{delinquencyRangeId}` — same envelope, but every field is optional.

### Create a bucket

`POST /v1/delinquency/buckets` (body type `DelinquencyBucketRequest`):

```json
{
  "name": "Basic bucket",
  "ranges": [1, 2, 3],
  "locale": "en"
}
```

## Response shapes

### Range

```json
{
  "id": 1,
  "classification": "Range 1-30",
  "minimumAgeDays": 1,
  "maximumAgeDays": 30
}
```

### Bucket

```json
{
  "id": 1,
  "name": "Basic bucket",
  "ranges": [
    { "id": 1, "classification": "Range 1-30", "minimumAgeDays": 1, "maximumAgeDays": 30 },
    { "id": 2, "classification": "Range 31-60", "minimumAgeDays": 31, "maximumAgeDays": 60 }
  ]
}
```

### Write responses

`CommandProcessingResult`:

```json
{ "resourceId": 7, "changes": { } }
```

## Permissions

Every handler checks `securityContext.authenticatedUser()` with one of `validateHasReadPermission`, `validateHasCreatePermission`, `validateHasUpdatePermission`, or `validateHasDeletePermission` against the single resource code `DELINQUENCY_BUCKET`.
