`ClientFamilyMembersApiResource` is a small CRUD surface for capturing household composition on a client — spouses, children, dependants. The records live in `m_family_members` and reference the parent client by `clientId`.

## Source

```
fineract-provider/src/main/java/org/apache/fineract/portfolio/client/api/ClientFamilyMembersApiResource.java
```

| Annotation | Value |
| --- | --- |

| `@Component` | yes |
| `@Tag` | `Client Family Member` |

Injected collaborators:

- `PlatformSecurityContext context`
- `ClientFamilyMembersReadPlatformService clientFamilyMembersReadPlatformService`
- `PortfolioCommandSourceWritePlatformService commandsSourceWritePlatformService`

## Permissions

Resource string: `FAMILYMEMBERS` (`RESOURCE_NAME_FOR_PERMISSIONS`). Reads use `validateHasReadPermission("FAMILYMEMBERS")`. Writes are gated by `CREATE_FAMILYMEMBERS`, `UPDATE_FAMILYMEMBERS`, `DELETE_FAMILYMEMBERS`.

## Endpoint inventory

| HTTP | Path | Description | Command / Read service |
| --- | --- | --- | --- |
| `GET` | `/v1/clients/{clientId}/familymembers` | List all family members of a client | `clientFamilyMembersReadPlatformService.getClientFamilyMembers(clientId)` |
| `GET` | `/v1/clients/{clientId}/familymembers/template` | Code-value dropdowns (relationship, gender, marital status, profession) | `clientFamilyMembersReadPlatformService.getFamilyMembersTemplate()` |
| `GET` | `/v1/clients/{clientId}/familymembers/{familyMemberId}` | Fetch one member | `clientFamilyMembersReadPlatformService.getClientFamilyMember(familyMemberId)` |

| `DELETE` | `/v1/clients/{clientId}/familymembers/{familyMemberId}` | Delete a member | `deleteFamilyMembers(familyMemberId)` |

## Source excerpt

```java
@Path("/v1/clients/{clientId}/familymembers")
@Component
@Tag(name = "Client Family Member")
public class ClientFamilyMembersApiResource {

    private static final String RESOURCE_NAME_FOR_PERMISSIONS = "FAMILYMEMBERS";

    @POST
    @Consumes(MediaType.APPLICATION_JSON)
    @Produces(MediaType.APPLICATION_JSON)
    public CommandProcessingResult addClientFamilyMembers(
            @PathParam("clientId") final long clientid,
            final String apiRequestBodyAsJson) {
        final CommandWrapper commandRequest = new CommandWrapperBuilder()
            .addFamilyMembers(clientid)
            .withJson(apiRequestBodyAsJson).build();
        return commandsSourceWritePlatformService.logCommandSource(commandRequest);
    }
}
```

## Read DTO

`org.apache.fineract.portfolio.client.data.ClientFamilyMembersData`:

```json
{
  "id": 3,
  "clientId": 187,
  "firstName": "Alex",
  "middleName": "",
  "lastName": "Doe",
  "qualification": "Spouse",
  "relationship": { "id": 1, "name": "Spouse", "position": 0 },
  "maritalStatus": { "id": 4, "name": "Married" },
  "gender": { "id": 16, "name": "Female" },
  "dateOfBirth": [1990, 1, 1],
  "profession": { "id": 12, "name": "Teacher" },
  "mobileNumber": "9999999999",
  "age": 32,
  "isDependent": false
}
```

The `/template` endpoint returns an empty `ClientFamilyMembersData` whose `relationshipIdOptions`, `genderIdOptions`, `maritalStatusIdOptions`, `professionIdOptions` arrays carry the code-value dropdowns sourced from `Relationship`, `Gender`, `MaritalStatus`, `Profession` code-value families.
