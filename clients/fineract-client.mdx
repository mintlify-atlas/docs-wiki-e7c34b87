`fineract-client` is the Retrofit2-based Java SDK for Apache Fineract. It is auto-generated by the `openapi-generator` Gradle plugin from `fineract.json` (the OpenAPI 3 spec produced by `fineract-provider:resolve`) using a customized Mustache template. The generated sources are wrapped by a hand-written `FineractClient` builder that wires OkHttp, basic auth, tenant headers, and TLS settings. The result is a fluent client where `client.clients().retrieveAll().execute()` produces a Retrofit `Response<T>`, or you can use the `Calls.ok(...)` helper to unwrap to `T` directly.

Source map:

- `fineract-client/build.gradle`
- `fineract-client/src/main/java/org/apache/fineract/client/util/FineractClient.java`
- `fineract-client/src/main/java/org/apache/fineract/client/util/Calls.java`
- `fineract-client/src/main/java/org/apache/fineract/client/util/CallFailedRuntimeException.java`
- `fineract-client/src/main/java/org/apache/fineract/client/util/adapter/ExternalIdAdapter.java`
- `fineract-client/src/main/java/org/apache/fineract/client/util/JSON.java`
- `fineract-client/src/main/resources/templates/java/api.mustache`

## Build configuration

The generation task lives in `fineract-client/build.gradle`:

```groovy
apply plugin: 'org.openapi.generator'
description = 'Fineract Client'

openApiMeta {
    generatorName = 'Fineract'
    packageName   = 'org.apache.fineract.client'
    outputFolder  = "$buildDir/meta".toString()
}

openApiValidate {
    inputSpec = "file:///$swaggerFile"
    recommend = true
}

openApiGenerate {
    outputDir = "$buildDir/generated/temp-java".toString()
}

task buildJavaSdk(type: org.openapitools.generator.gradle.plugin.tasks.GenerateTask) {
    generatorName    = 'java'
    verbose          = false
    validateSpec     = false
    skipValidateSpec = true
    inputSpec        = "file:///$swaggerFile"
    outputDir        = "$buildDir/generated/temp-java".toString()
    templateDir      = "$projectDir/src/main/resources/templates/java"
    groupId          = 'org.apache.fineract'
    apiPackage       = 'org.apache.fineract.client.services'
    invokerPackage   = 'org.apache.fineract.client'
    modelPackage     = 'org.apache.fineract.client.models'
    configOptions = [
        dateLibrary            : 'java8',
        useRxJava2             : 'false',
        library                : 'retrofit2',
        hideGenerationTimestamp: 'true',
        containerDefaultToNull : 'true',
        oauth2Implementation   : 'none'
    ]
    generateModelTests   = false
    generateApiTests     = false
    ignoreFileOverride   = "$projectDir/.openapi-generator-ignore"
    dependsOn(':fineract-provider:resolve')
}
```

The interesting `configOptions`:

| Option                      | Value         | Effect                                                                                                                                                |
|-----------------------------|---------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| `dateLibrary`               | `java8`       | Uses `java.time.LocalDate` / `OffsetDateTime` in models.                                                                                               |
| `useRxJava2`                | `false`       | No reactive return types; each API method returns `retrofit2.Call<T>`.                                                                                |
| `library`                   | `retrofit2`   | Generates Retrofit interfaces (`@GET`, `@POST`, `@Body`, …).                                                                                         |
| `hideGenerationTimestamp`   | `true`        | Strips the `@Generated(value="…", date="…")` line so diffs are stable across runs.                                                                    |
| `containerDefaultToNull`    | `true`        | Lists/maps default to `null` instead of empty — matters because the server distinguishes "absent" from "empty" in some endpoints.                       |
| `oauth2Implementation`      | `none`        | Don't generate the buggy oltu `OAuthOkHttpClient`; users plug in their own interceptor (the generated file is in fact explicitly deleted, see below). |

### Post-generation cleanup

```groovy
task cleanupGeneratedJavaFiles() {
    def tempDir   = file("$buildDir/generated/temp-java")
    def targetDir = file("$buildDir/generated/java")

    doFirst {
        delete fileTree(tempDir) {
            include "src/main/java/org/apache/fineract/client/auth/OAuthOkHttpClient.java"
        }
        delete fileTree(targetDir) {
            include "src/main/java/org/apache/fineract/client/auth/OAuthOkHttpClient.java"
        }
    }
    doLast {
        copy {
            from tempDir
            into targetDir
            filter { line ->
                line.replaceAll("import org\\.joda\\.time\\.\\*;", "")
                    .replaceAll(", \\)", ")")
                    .replaceAll(", , @HeaderMap", ", @HeaderMap")
                    .replaceAll("\\(, ", "(")
            }
            duplicatesStrategy = DuplicatesStrategy.EXCLUDE
        }
    }
    dependsOn("buildJavaSdk")
}
```

Three workarounds:

1. **`OAuthOkHttpClient.java` deleted** — the openapi-generator scaffolds an obsolete `oltu`-based OAuth client even with `oauth2Implementation: none`; we just nuke it.
2. **Strip stray `import org.joda.time.*`** — left over from an older spec.
3. **Fix `, )` / `(, ` / `, , @HeaderMap`** — corner cases when the Mustache template emits empty parameter slots.

`compileJava` then depends on the cleaned-up `build/generated/java/...` tree:

```groovy
sourceSets {
    main {
        java {
            srcDir new File(buildDir, "generated/java/src/main/java")
            destinationDirectory = layout.buildDirectory.dir('classes/java/main').get().asFile
        }
    }
}

tasks.named('compileJava') {
    dependsOn(buildJavaSdk, buildTypescriptAngularSdk, buildAsciidoc,
              cleanupGeneratedJavaFiles, licenseFormatMain, spotlessMiscApply)
}
```

`compileJava` ordering matters: license formatting runs before compile so the inserted ASF header doesn't break Spotless.

A generated API interface looks like (excerpt):

```java
public interface ClientApi {
    @GET("v1/clients")
    Call<GetClientsResponse> retrieveAll21(
            @Query("officeId") Long officeId,
            @Query("externalId") String externalId,
            @Query("displayName") String displayName,
            @Query("firstName") String firstName,
            @Query("lastName") String lastName,
            @Query("status") String status,
            @Query("underHierarchy") String underHierarchy,
            @Query("offset") Integer offset,
            @Query("limit") Integer limit,
            @Query("orderBy") String orderBy,
            @Query("sortOrder") String sortOrder,
            @Query("orphansOnly") Boolean orphansOnly,
            @Query("sqlSearch") String sqlSearch
    );

    @POST("v1/clients")
    @Headers({ "Content-Type:application/json" })
    Call<PostClientsResponse> create7(@Body PostClientsRequest postClientsRequest);

    @GET("v1/clients/{clientId}")
    Call<GetClientsClientIdResponse> retrieveOne8(@Path("clientId") Long clientId,
            @Query("staffInSelectedOfficeOnly") Boolean staffInSelectedOfficeOnly);

    // … plus ~50 other endpoints under /v1/clients/*
}
```

Note the method names — `retrieveAll21`, `create7` — the suffixes come from the openapi-generator's deduplication when multiple operations share a name across the spec. The actual generated names are stable per build because `sortOutput = true` in `resolve { … }`.

## FineractClient builder (hand-written wrapper)

The auto-generated code provides the API interfaces. The hand-written `org.apache.fineract.client.util.FineractClient` (in `fineract-client/src/main/java/...`) gives you the convenient builder and per-API accessors:

```java
public final class FineractClient {

    public static final String DATE_FORMAT = "yyyy-MM-dd";

    // Each of these is a Retrofit-built API client
    public final ClientApi clients;
    public final LoansApi loans;
    public final SavingsAccountApi savingsAccount;
    public final GroupsApi groups;
    public final AccountingClosureApi accountingClosure;
    // … 100+ more fields

    private final OkHttpClient okHttpClient;
    private final Retrofit retrofit;

    private FineractClient(OkHttpClient okHttpClient, Retrofit retrofit) {
        this.okHttpClient = okHttpClient;
        this.retrofit = retrofit;
        clients         = retrofit.create(ClientApi.class);
        loans           = retrofit.create(LoansApi.class);
        savingsAccount  = retrofit.create(SavingsAccountApi.class);
        // … one create() per API
    }

    public static Builder builder() { return new Builder(); }

    public OkHttpClient okHttpClient() { return okHttpClient; }
    public HttpUrl       baseURL()     { return retrofit.baseUrl(); }

    public <S> S createService(Class<S> serviceClass) {
        return retrofit.create(serviceClass);
    }

    public static final class Builder {
        private final JSON json = new JSON();
        private final OkHttpClient.Builder okBuilder = new OkHttpClient.Builder();
        private final Retrofit.Builder retrofitBuilder = new Retrofit.Builder()
                .addConverterFactory(ScalarsConverterFactory.create())
                .addConverterFactory(GsonCustomConverterFactory.create(json.getGson()));

        private String baseURL;
        private String tenant;
        private String username;
        private String password;

        public Builder baseURL(String baseURL)       { this.baseURL = baseURL;  return this; }
        public Builder tenant (String tenant)        { this.tenant  = tenant;   return this; }
        public Builder basicAuth(String u, String p) { this.username = u; this.password = p; return this; }

        public Builder logging(Level level) {
            HttpLoggingInterceptor logging = new HttpLoggingInterceptor();
            logging.setLevel(level);
            okBuilder.addInterceptor(logging);
            return this;
        }

        public Builder readTimeout(Duration timeout) { okBuilder.readTimeout(timeout); return this; }

        public Builder insecure(boolean insecure) {
            if (insecure) {
                // configure an all-trust X509TrustManager and a permissive HostnameVerifier
                // — DO NOT USE in production
            }
            return this;
        }

        public FineractClient build() {
            retrofitBuilder.baseUrl(has("baseURL", baseURL));

            if (tenant != null) {
                ApiKeyAuth tenantAuth = new ApiKeyAuth("header", "fineract-platform-tenantid");
                tenantAuth.setApiKey(has("tenant", tenant));
                okBuilder.addInterceptor(tenantAuth);
            }

            if (username != null && password != null) {
                HttpBasicAuth basicAuth = new HttpBasicAuth();
                basicAuth.setCredentials(has("username", username), has("password", password));
                okBuilder.addInterceptor(basicAuth);
            }

            OkHttpClient okHttpClient = okBuilder.build();
            retrofitBuilder.client(okHttpClient);
            return new FineractClient(okHttpClient, retrofitBuilder.build());
        }
    }
}
```

The builder methods:

| Method                                 | Effect                                                                                                |
|----------------------------------------|-------------------------------------------------------------------------------------------------------|
| `baseURL(String)`                      | Sets `Retrofit.Builder.baseUrl(...)` — typically `https://server/fineract-provider/api/`.            |
| `tenant(String)`                       | Adds an `ApiKeyAuth` interceptor that sends `fineract-platform-tenantid: <value>` on every request.   |
| `basicAuth(String, String)`            | Adds an `HttpBasicAuth` interceptor.                                                                  |
| `logging(Level)`                       | Wraps an `HttpLoggingInterceptor` with the chosen verbosity.                                          |
| `readTimeout(Duration)`                | Forwards to `OkHttpClient.Builder.readTimeout`.                                                       |
| `insecure(boolean)`                    | Installs an all-trusting `X509TrustManager` and `HostnameVerifier`. Documented as **not for prod**. |
| `build()`                              | Materializes the OkHttp client + Retrofit instance, then constructs the `FineractClient`.            |

For advanced customization (custom interceptors, alternative converter factories) call `retrofitBuilder()` / `okBuilder()` (omitted from the snippet above) and finally `build()`.

## Calls helper

Retrofit's call style requires you to call `.execute()` and then check `Response<T>.isSuccessful()`. The `Calls` helper hides that:

```java
public final class Calls {

    public static <T> T ok(Call<T> call) {
        try {
            Response<T> response = call.execute();
            if (!response.isSuccessful()) {
                throw new CallFailedRuntimeException(response);
            }
            return response.body();
        } catch (IOException e) {
            throw new CallFailedRuntimeException(e);
        }
    }

    public static <T> Optional<T> okOptional(Call<T> call) {
        return Optional.ofNullable(ok(call));
    }
}
```

`CallFailedRuntimeException` captures the HTTP status, headers, and body of the failed response so you can inspect them later.

## ExternalIdAdapter

`fineract-client/src/main/java/org/apache/fineract/client/util/adapter/ExternalIdAdapter.java` is a Gson `TypeAdapter` that translates between the platform's `ExternalId` JSON representation (an object `{ "value": "abc-123" }`) and the SDK's simple `String` field. It is registered in `JSON` and is what makes round-trips of external IDs work transparently.

## End-to-end usage

```java
FineractClient fineract = FineractClient.builder()
    .baseURL("https://localhost:8443/fineract-provider/api/")
    .tenant("default")
    .basicAuth("mifos", "password")
    .insecure(true)
    .logging(HttpLoggingInterceptor.Level.HEADERS)
    .readTimeout(Duration.ofSeconds(30))
    .build();

// 1. Retrieve a single client
GetClientsClientIdResponse client =
        Calls.ok(fineract.clients.retrieveOne8(1L, false));
System.out.println("Display name: " + client.getDisplayName());

// 2. Create a new client
PostClientsRequest req = new PostClientsRequest();
req.setFirstname("Alice");
req.setLastname("Mwangi");
req.setOfficeId(1L);
req.setDateFormat("dd MMMM yyyy");
req.setLocale("en");
req.setActive(false);

PostClientsResponse created = Calls.ok(fineract.clients.create7(req));
System.out.println("Created client " + created.getClientId());

// 3. List loans for that client
GetLoansResponse loans = Calls.ok(
    fineract.loans.retrieveAll27(null, null, "clientId," + created.getClientId(),
            null, null, null, null, null, null, null, null));
```

## Regenerating after a server change

When `fineract-provider`'s JAX-RS surface changes:

```bash
# 1. Regenerate the OpenAPI spec
./gradlew :fineract-provider:resolve

# 2. Regenerate the SDK
./gradlew :fineract-client:buildJavaSdk --rerun-tasks

# 3. Recompile
./gradlew :fineract-client:compileJava

# 4. Optionally publish locally for downstream consumers
./gradlew :fineract-client:publishToMavenLocal
```

If you want to inspect the diff before regenerating:

```bash
jq -r '.paths | keys[]' fineract-provider/build/resources/main/static/fineract.json | sort > /tmp/new.txt
jq -r '.paths | keys[]' fineract-provider/config/swagger/fineract-baseline.json    | sort > /tmp/old.txt
diff /tmp/old.txt /tmp/new.txt
```

## Other generators wired in the same module

`fineract-client/build.gradle` defines three additional tasks driven by the same spec:

| Task                            | Output                                                                       | Purpose                                              |
|---------------------------------|------------------------------------------------------------------------------|------------------------------------------------------|
| `buildTypescriptAngularSdk`     | `build/generated/typescript/apache-fineract-client/...`                      | Angular npm package `@apache/fineract-client`.       |
| `buildAsciidoc`                 | `build/generated/asciidoc/...`                                               | Reference docs source for the Apache site.           |
| `buildJavaSdk`                  | `build/generated/temp-java` → cleaned `build/generated/java`                  | The Java SDK described on this page.                  |

The TypeScript generator uses ng12-style metadata (`ngVersion: '12.0.0'`) — bump in `configOptions` if you need a newer Angular target.

## Mustache template override (`api.mustache`)

`fineract-client/src/main/resources/templates/java/api.mustache` overrides the openapi-generator default. Key edits:

- Adds an ASF license header.
- Replaces the generic `@SuppressWarnings("unused")` with project-specific ones.
- Tidies imports.
- Inserts the `JSON` configuration logic for `ExternalIdAdapter`.

If you need to surface custom imports in every generated API, this file is the lever.

## Common pitfalls

| Symptom                                                                          | Cause                                                                                              | Fix                                                                                                                |
|----------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| `404` on every call                                                              | `baseURL` missing the `/fineract-provider/api/` suffix; or missing trailing slash                  | Use `https://host:8443/fineract-provider/api/` (the slash matters for Retrofit's URL composition)                  |
| `400` with `"locale": missing`                                                   | Most write endpoints require `locale` and `dateFormat` in the body                                  | Set them on every request DTO: `req.setLocale("en"); req.setDateFormat("yyyy-MM-dd")`                              |
| `SSLHandshakeException: PKIX path building failed`                              | Server uses a self-signed cert                                                                      | Local dev only: `.insecure(true)` (production: add cert to JVM truststore)                                          |
| `CallFailedRuntimeException` with empty body                                     | Server returned a JSON problem detail but Retrofit logged the body as gone                         | Inspect `e.response().errorBody().string()` in a clone; or enable `Level.BODY`                                      |
| Inconsistent enum values across builds                                          | Spec change introduced a new enum value; regenerated SDK loaded against old server                  | Pin SDK version to server version                                                                                   |
| Method name suddenly renamed (e.g. `retrieveAll21` → `retrieveAll22`)            | A new endpoint shifted the dedup numbering                                                          | Address by tag + path: write a small adapter layer that hides the generated names                                   |
| `ClassNotFoundException: org.joda.time.LocalDate`                                | Spec mistakenly still has joda type references after a generator change                            | Re-run `cleanupGeneratedJavaFiles` (already wired); confirm the post-filter strips `import org.joda.time.*`         |
